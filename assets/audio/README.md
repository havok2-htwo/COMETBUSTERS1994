Lege WAV-Dateien in die passenden Unterordner.

Beispiel:
- `sounds/laser_fire/laser01.wav`
- `sounds/laser_fire/laser02.wav`
- `sounds/explosion_large/boom01.wav`

Beim Start werden alle `.wav`-Dateien je Event-Ordner geladen und spaeter zufaellig abgespielt.
Wenn ein Ordner leer bleibt, faellt das Spiel automatisch auf die eingebauten Browser-Fallback-Sounds zurueck.
